# Projeto para estudo de criação de APP TUI

Este projeto foi criado durante o estudo da criação de aplicações TUI feitas em python com a biblioteca [Textual](https://textual.textualize.io/)

## Pré Requisitos para Instalação

Instalar:

1. Python >= 3.13.7
2. Pip

## Instalação

1. clonar respositorio

2. acessar a pasta do projeto utilizando o comando `cd`

3. Entrar no ambiente virtual

        source first_tui_project_env/bin/activate

4. Instalar bibliotecas

        pip install

## Comandos uteis

Para desativar espaço virtual

        deactivate
